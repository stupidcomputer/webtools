When you're doing a task, it's common to come across commonly repeated patterns and actions. Usually when this happens to me, I write a program to automate this stuff away. Most of the time, though, they're terminal programs, so they're hard for other people to use. I'd like to learn web technologies and how to website (tm) and so this is the impetus for this project.
