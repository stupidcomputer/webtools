Generic utilities

  • A simple presentation timer. Shows only minutes with black-on-white text. Made for use with a projector or screen in the back of an auditorium.
  • A visualizer for body vitals. Uses a bespoke JSON format that my org-mode abomination outputs. Good luck using this?

Flag program utilities

For more marching band and most broadly music-related stuff, go to marching.beepboop.systems.

  • A flag program volunteer visualizer. Shows volunteer coverage of flag holidays based on SignUpGenius output.
  • A flag program route generator. Generates route sheets for flag distributors. Automagically calculates page breaks with HTML magic and allows some easy modification.
  • A flag program volunteer compliance tool. Outputs a handy CSV that contains some boiled down stats for visualizing compliance issues.
